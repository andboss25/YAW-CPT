# YET ANOTHER WEB CRAWLER PENTESTING TOOL
## YAW-CPT
**by andreiplsno**

## This project is under development and you may encounter bugs!

YAW-CPT is a penetration testing tool designed to automatically detect the nature of links before crawling trough them.
It is also designed to categorize links based on factors such as if its a css link, an api link, exxternal link, etc...

Sadly this project is quite new and we (just me right now) need to implement more features.

Current and planed features:
- Extract and crawl all links from a page (including ones found in scripts) -> implemented
- TOR support -> implemented
- basic categorization -> implemented
- scripts for categorization and detection -> TODO: implement
